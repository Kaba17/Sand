## Packages
framer-motion | Smooth animations for page transitions and wizard steps
date-fns | Date formatting with Arabic locale support
react-dropzone | Drag and drop file uploads
clsx | Conditional class names utility
tailwind-merge | Utility for merging tailwind classes

## Notes
The application requires Right-to-Left (RTL) layout for Arabic support.
Ensure "Cairo" and "Tajawal" fonts are loaded.
File uploads should use multipart/form-data to /api/upload (implied endpoint).
Authentication uses Replit Auth (useAuth hook).
